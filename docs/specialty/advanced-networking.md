## Hands-on Practice

### AWS Free Tier

The AWS Free Tier covers networking services: VPC (free), Route 53 (50 hosted zones), CloudFront (1 TB transfer), and Network Firewall (free tier). Build complex network topologies, configure DNS routing, and set up hybrid connectivity. Set up billing alerts and monitor usage to avoid unexpected charges.

[AWS Free Tier](https://aws.amazon.com/free/)

### AWS Builder Center Sandbox (Official)

- Official free AWS sandbox environment

- No risk of accidental AWS charges

https://aws.amazon.com/about-aws/whats-new/2026/07/aws-builder-center-sandbox/

### AWS Builder Labs

[AWS Builder Labs](https://aws.amazon.com/training/digital/aws-builder-labs/)

### AWS Skill Builder Labs

Skill Builder offers additional networking labs covering Transit Gateway, Direct Connect, Network Firewall, and advanced VPC features.

[Skill Builder Labs](https://skillbuilder.aws/labs)

### Networking Services to Practice

- **Amazon VPC** — Design multi-tier VPCs with public/private subnets, NAT gateways, VPC endpoints, and peering.
- **AWS Transit Gateway** — Connect multiple VPCs and on-premises networks with centralized routing.
- **Amazon Route 53** — Configure routing policies (geolocation, latency, failover), private hosted zones, and DNS resolution.
- **AWS Direct Connect** — Understand virtual interfaces, link aggregation, and failover configurations.
- **AWS Network Firewall** — Build stateful firewall rules and intrusion prevention policies.
- **Elastic Load Balancing** — Configure cross-zone load balancing, connection draining, and stickiness.

### Third-Party Platforms

- **Cloud Academy** — Structured networking deep-dives with hands-on labs.

## FAQ

**Do I need all this material?**
Mix up as much as you can. This is one of the toughest AWS exams unless you have a networking background.

**Do I need hands-on work?**
Yes, but advanced network scenarios may not be possible on sandboxes.

**Where can I find vouchers?**
See [Vouchers & Discounts](../vouchers-discounts.md). Usually no Specialty discounts. Passing SAA gives 50% off ANS.

**Can I cheat using dumps?**
High risk, not worth it.

**Can I pass with only free resources?**
Not many free resources exist. Spending on quality courses and practice exams is strongly recommended.

**Can ANS be my first AWS cert?**
No. You need hands-on AWS experience (SAA/SA Pro level).

**Are there books for this exam?**
Books are largely out of date. A Sybex guide exists but video courses are recommended instead.

**Should I buy TutorialsDojo via Udemy?**
Not available. Buy direct from their site.

**I failed my practice exams — is that normal?**
Yes. Review incorrect answers and learn from them.

**What score should I get on practice exams?**
There is no magic number. The learning from reviewing incorrect answers matters more than the score.

**Can I skip a topic if someone said it wasn't on their exam?**
No. Everyone gets a different exam from a large question pool. Study everything in the exam guide.

# AWS Cloud Institute

!!! warning "Program Ending June 30, 2026"
    AWS Cloud Institute will conclude on **June 30, 2026**, with Q2 2026 as the final cohort. Applications for the final cohort are no longer being accepted. Visit [AWS Skill Builder](https://skillbuilder.aws/) for 900+ cloud and AI courses as an alternative.

---

## What Is AWS Cloud Institute?

AWS Cloud Institute is a **structured, virtual training program** built by AWS experts to help individuals go from non-technical backgrounds to cloud developers in as little as 9 months. The program combines self-paced digital courses, instructor-led sessions, hands-on labs, capstone projects, career services, and certification preparation.

Originally launched in 2023, the program was designed to address the growing demand for cloud developers by providing a comprehensive learning pathway with industry-recognized credentials.

---

## Who Is It For?

AWS Cloud Institute is designed for:

- Individuals with **no prior technical experience** looking to start a career in cloud development
- Career changers transitioning from non-tech fields
- Self-motivated learners who prefer a structured, cohort-based program
- Anyone seeking **entry-level cloud developer roles**

!!! note "Eligibility Requirements"
    - Must be **18 years or older**
    - Must hold a **high school diploma or equivalent**
    - Must have a **U.S. billing address** (program was U.S.-only)
    - Must complete an application and be invited to enroll

---

## Curriculum

The program consists of **9 digital training courses** delivered over 3 quarters, supported by:

- **150+ interactive labs**
- **Live and on-demand instructor-led sessions**
- **2 capstone projects**
- **Portfolio development**
- **Career services and professional skill development**

### Quarter 1

| Course | Focus |
|--------|-------|
| Introduction to Cloud Foundations | Core cloud concepts, AWS services overview |
| Developer Fundamentals | Programming basics, Python, development tools |
| AWS Cloud Operations 1 | Compute, storage, networking fundamentals |

**Certification prep:** AWS Certified Cloud Practitioner

### Quarter 2

| Course | Focus |
|--------|-------|
| AWS Cloud Fundamentals 1 | Architecting, databases, security |
| AI for Developers | Machine learning, generative AI, Amazon Bedrock |
| Developer Intermediate 1 | Application development, CI/CD, DevOps basics |

**Certification prep:** AWS Certified AI Practitioner
**Capstone:** Cloud Capstone 1

### Quarter 3

| Course | Focus |
|--------|-------|
| AWS Cloud Fundamentals 2 | Advanced architecture, migration, hybrid cloud |
| AWS Cloud Operations 2 | Monitoring, automation, incident response |
| Developer Intermediate 2 | Advanced development, CDK, serverless |

**Certification prep:** AWS Certified Developer Associate
**Capstone:** Cloud Capstone 2

## Certifications Earned

AWS Cloud Institute prepares you for **three AWS certifications**, with exam vouchers provided upon completion of prerequisite courses:

| Certification | Level | Prerequisite Course |
|---------------|-------|---------------------|
| AWS Certified Cloud Practitioner (CLF-C02) | Foundational | Introduction to Cloud Foundations |
| AWS Certified AI Practitioner (AIF-C01) | Foundational | AI for Developers |
| AWS Certified Developer Associate (DVA-C02) | Associate | Developer Intermediate 2 |

## AWS Cloud Institute Learner Grants

AWS Cloud Institute offered partial and full **Learner Grants** to reduce cost barriers for select learners with financial constraints.

### Grant Types

| Grant | Coverage | Learner Pays |
|-------|----------|--------------|
| **Full Grant** | 100% of course fees | $1/month + sales taxes |
| **Partial Grant** | 50% of course fees | 50% + sales taxes |

### Eligibility

- Demonstrated financial need
- Accepted into AWS Cloud Institute
- U.S. residency (program was U.S.-only)

### Application Process

1. Complete AWS Cloud Institute application
2. Indicate interest in financial assistance during the application process
3. Submit required financial documentation
4. Grant decision communicated before enrollment deadline

Grants were automatically applied to monthly invoices for the entire enrollment period.

!!! tip
    As the program is concluding, Learner Grants are no longer available for new applicants.

## Career Outcomes

AWS Cloud Institute was designed with input from employers to ensure graduates have the skills needed for **entry-level cloud developer roles**. The program included:

- **Skills mapping** validated by industry employers
- **Career services** including resume review, interview coaching, and job search strategies
- **Portfolio development** with real capstone projects
- **Networking opportunities** with peers and AWS professionals

Graduates receive an **AWS Cloud Institute program certificate** and a **digital badge** verifying program completion.

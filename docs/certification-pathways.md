## Decision Matrix

| Your Background | Recommended First Cert | Why |
|---|---|---|
| Non-technical, new to cloud | **CLF-C02** Cloud Practitioner | Teaches cloud concepts without assuming technical skills |
| Student (any field) | **CLF-C02** Cloud Practitioner | Foundational, low barrier to entry |
| IT professional (sysadmin, network, support) | **SAA-C03** Solutions Architect Associate | Broadest coverage; skip CLF |
| Software developer | **SAA-C03** then **DVA-C02** Developer Associate | Build the foundation first, then specialize |
| Data / analytics professional | **SAA-C03** then **DEA-C01** Data Engineer Associate | SAA provides the AWS fundamentals DEA builds on |
| AI/ML practitioner | **SAA-C03** then **AIF-C01** AI Practitioner or **MLA-C01** ML Engineer Associate | SAA gives the infrastructure context ML certs assume |
| Security professional | **SAA-C03** then **SCS-C02** Security Specialty | SAA covers security fundamentals; SCS goes deep |
| Manager / non-technical leader | **CLF-C02** Cloud Practitioner | Broad overview without deep technical detail |
| Experienced cloud professional | **SAA-C03** then Professional or Specialty | Skip CLF, validate your existing knowledge |

### Choose Your Level

| If You... | Start At |
|---|---|
| Have 0–6 months of IT/cloud experience | Foundational (CLF or AIF) |
| Have 6–12 months of general IT experience | Associate (SAA recommended) |
| Have 1–2+ years of hands-on AWS experience | Associate (SAA), skip CLF |
| Have 2+ years of AWS and want advanced validation | Professional or Specialty (after SAA) |

## Individual Pathways

### Absolute Beginner

Someone with little to no IT or cloud experience who wants to start a career in cloud computing.

**Certification Order:** 1. [CLF-C02](foundational/cloud-practitioner.md) → 2. [AIF-C01](foundational/ai-practitioner.md) (optional) → 3. [SAA-C03](associate/solutions-architect.md)

**Timeline:** 3–6 months

**Next:** [Data Engineer](#data-engineer), [Developer](#developer), or [Cloud Engineer](#cloud-engineer)

### Student

Currently enrolled in school, college, or university looking to build cloud skills alongside your studies.

**Certification Order:** 1. [CLF-C02](foundational/cloud-practitioner.md) → 2. [SAA-C03](associate/solutions-architect.md) → 3. [DVA-C02](associate/developer.md) (if pursuing development)

**Timeline:** 2–4 months per certification

**Next:** [Developer](#developer) or [Cloud Engineer](#cloud-engineer)

### Developer

Software developer looking to validate AWS development skills.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [DVA-C02](associate/developer.md) → 3. [DOP-C02](professional/devops-engineer-pro.md)

**Timeline:** 2–3 mo (SAA) + 2–3 mo (DVA) + 3–4 mo (DOP)

**Related:** [DevOps Engineer](#devops-engineer), [Platform Engineer](#platform-engineer)

### DevOps Engineer

Operations-focused professional looking to master CI/CD, automation, and infrastructure as code.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [SOA-C03](associate/cloudops.md) → 3. [DOP-C02](professional/devops-engineer-pro.md)

**Timeline:** 6–9 months total

**Related:** [Developer](#developer), [Cloud Engineer](#cloud-engineer)

### Solutions Architect

IT professional aiming to design and implement scalable, fault-tolerant systems on AWS.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [SAP-C02](professional/solutions-architect-pro.md)

**Timeline:** 3–4 mo (SAA) + 4–6 mo (SAP)

**Related:** [Security Engineer](#security-engineer), [Networking Engineer](#networking-engineer)

### SysOps Administrator

System administrator managing AWS operations, monitoring, and incident response.

**Certification Order:** 1. [CLF-C02](foundational/cloud-practitioner.md) (optional) → 2. [SOA-C03](associate/cloudops.md) → 3. [DOP-C02](professional/devops-engineer-pro.md)

!!! note "Exam Rename"
    SOA-C02 was previously called SysOps Administrator Associate. It has been renamed to **CloudOps Engineer Associate (SOA-C03)**.

**Timeline:** 2–3 mo (SOA) + 3–4 mo (DOP)

**Related:** [Cloud Engineer](#cloud-engineer), [DevOps Engineer](#devops-engineer)

### Data Engineer

Data professional working with data pipelines, storage, and analytics.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [DEA-C01](associate/data-engineer.md)

**Timeline:** 2–3 mo (SAA) + 2–4 mo (DEA)

**Related:** [Database Engineer](#database-engineer)

### Machine Learning Engineer

AI/ML practitioner building, training, and deploying ML models on AWS.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [AIF-C01](foundational/ai-practitioner.md) (optional) → 3. [MLA-C01](associate/machine-learning.md)

**Timeline:** 2–3 mo (SAA) + 2–3 mo (MLA)

**Related:** [AI Engineer](#ai-engineer)

### AI Engineer

Professional focused on building generative AI applications.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [AIF-C01](foundational/ai-practitioner.md) → 3. [MLA-C01](associate/machine-learning.md) → 4. [AIP-C01](professional/genai-developer-pro.md)

**Timeline:** 5–8 months total

**Related:** [Machine Learning Engineer](#machine-learning-engineer)

### Security Engineer

Security professional focused on securing AWS environments.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [SCS-C02](specialty/security.md)

!!! tip "SAA first"
    SAA covers foundational security concepts (IAM, encryption, compliance) that SCS builds on. AWS recommends Associate-level knowledge before attempting Specialty exams.

**Timeline:** 3–4 mo (SAA) + 2–3 mo (SCS)

**Related:** [Solutions Architect](#solutions-architect)

### Networking Engineer

Network professional managing VPCs, hybrid connectivity, and DNS.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [ANS-C01](specialty/advanced-networking.md)

!!! warning "ANS-C01 Retiring August 25, 2026"
    The Advanced Networking Specialty exam is being retired. Certifications earned before retirement remain active for 3 years.

**Timeline:** 3–4 mo (SAA) + 2–3 mo (ANS)

**Related:** [Security Engineer](#security-engineer)

### Database Engineer

Database professional specializing in AWS database services.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [DEA-C01](associate/data-engineer.md)

!!! tip "DEA covers database topics"
    While there is no dedicated Database Specialty certification, the Data Engineer Associate exam covers database design, migration, and management extensively.

**Timeline:** 3–4 mo (SAA) + 2–3 mo (DEA)

**Related:** [Data Engineer](#data-engineer), [Solutions Architect](#solutions-architect)

### Cloud Engineer

Generalist cloud professional managing cloud infrastructure and operations.

**Certification Order:** 1. [CLF-C02](foundational/cloud-practitioner.md) (optional) → 2. [SAA-C03](associate/solutions-architect.md) → 3. [SOA-C03](associate/cloudops.md) → 4. [DOP-C02](professional/devops-engineer-pro.md)

**Timeline:** 6–12 months total

**Related:** [DevOps Engineer](#devops-engineer), [SysOps Administrator](#sysops-administrator)

### Platform Engineer

Engineer building and maintaining internal developer platforms on AWS.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [DVA-C02](associate/developer.md) → 3. [DOP-C02](professional/devops-engineer-pro.md)

**Timeline:** 6–9 months total

**Related:** [Developer](#developer), [DevOps Engineer](#devops-engineer)

### FinOps

Finance or operations professional focused on cloud cost management and optimization.

**Certification Order:** 1. [CLF-C02](foundational/cloud-practitioner.md) → 2. [SAA-C03](associate/solutions-architect.md)

**Timeline:** 2–3 months per certification

**Related:** [Manager](#manager)

### Technical Trainer

Educator or trainer delivering AWS training content.

**Certification Order:** 1. [CLF-C02](foundational/cloud-practitioner.md) → 2. [SAA-C03](associate/solutions-architect.md) → 3. Additional based on training focus

**Timeline:** Ongoing

**Related:** All pathways — trainers benefit from breadth over depth

### Consultant

AWS consultant delivering solutions to clients across multiple industries.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [SAP-C02](professional/solutions-architect-pro.md) → 3. [SCS-C02](specialty/security.md) (strongly recommended)

**Timeline:** 6–12 months to SA Pro + Security

**Related:** [Solutions Architect](#solutions-architect), [Security Engineer](#security-engineer)

### Manager

Engineering or IT manager overseeing cloud teams and strategy.

**Certification Order:** 1. [CLF-C02](foundational/cloud-practitioner.md) → 2. [SAA-C03](associate/solutions-architect.md)

**Timeline:** 2–3 months per certification

**Related:** [FinOps](#finops)

### Experienced Professional

Veteran IT professional looking to formalize AWS expertise.

**Certification Order:** 1. [SAA-C03](associate/solutions-architect.md) → 2. [SAP-C02](professional/solutions-architect-pro.md) → 3. [DOP-C02](professional/devops-engineer-pro.md) → 4. [SCS-C02](specialty/security.md)

**Timeline:** 6–12 months for all certs

**Related:** [Consultant](#consultant), [Solutions Architect](#solutions-architect)

### Career Changer

Professional transitioning from a non-tech field into cloud computing.

**Certification Order:** 1. [CLF-C02](foundational/cloud-practitioner.md) → 2. [SAA-C03](associate/solutions-architect.md) → 3. [DVA-C02](associate/developer.md) (if pursuing development)

**Timeline:** 6–12 months for 2–3 certifications

**Related:** [Absolute Beginner](#absolute-beginner), [Student](#student)

## FAQ

### Which certification should I start with?

For most people, **Solutions Architect Associate (SAA-C03)** is the best starting point. If you have no IT background, start with **Cloud Practitioner (CLF-C02)** first.

### Should I skip Cloud Practitioner?

If you have 6+ months of IT experience, you can skip CLF and start directly with SAA. Cloud Practitioner is optional.

### What order should I take certifications?

1. Solutions Architect Associate (SAA)
2. Choose a specialization: Developer, CloudOps, Data Engineer, ML Engineer, or Security
3. Professional level: DevOps Engineer Professional or Solutions Architect Professional
4. Specialty certifications as needed

### How many certifications do I need?

One well-chosen certification is more valuable than many poorly-prepared ones. Most professionals benefit from 2–4 certifications.

### Do certifications expire?

Yes — all AWS certifications are valid for **3 years**. Passing a higher-level certification renews active certifications below it.

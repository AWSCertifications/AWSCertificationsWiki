# Free AWS Learning Resources

A comprehensive guide to free AWS learning resources, organized by skill level.

> **Tip:** Always check the official AWS website for the latest free offerings, as new resources are regularly added.

## Beginner Level

Resources that require **no payment** to access.

- **AWS Skill Builder** – Free digital courses, learning plans, and exam prep.  
  <https://skillbuilder.aws/>

- **AWS Educate** – Free access to AWS content, hands-on labs, and career pathways for students and educators.

- **AWS re:Post** – A free community-driven knowledge sharing platform where AWS experts answer your questions.

- **AWS Documentation & Whitepapers** – Official docs, architecture guidance, and well-architected whitepapers, all free.

- **AWS Free Tier** – Hands-on practice with many AWS services at no cost for 12 months (plus always-free and trial offers).

- **YouTube Channels** – AWS Official, re:Invent session recordings, and AWS Events playlist.

- **FreeCodeCamp / Andrew Brown** – Full-length AWS certification courses on YouTube (e.g., Exam Readiness, Cloud Practitioner, Solutions Architect).

- **Digital Badges** – AWS Knowledge Badges; most are free and can be earned through short assessments.

## Intermediate Level

Resources that are **free to use** but typically require an **active AWS account**.

> **Warning:** Some workshops and labs may incur small costs if you exceed the Free Tier limits. Always clean up resources after use.

- **AWS Workshops** – Curated workshop guides at <https://workshops.aws/>. Step-by-step, topic-specific, and free.

- **AWS Builder Center Sandbox** – Official free temporary AWS environment. No account or credit card required. Sessions up to 8 hours with automatic cleanup.  
  <https://aws.amazon.com/about-aws/whats-new/2026/07/aws-builder-center-sandbox/>

- **AWS Builder Labs** – Free, guided labs within AWS Skill Builder that provide temporary AWS accounts.  
  <https://aws.amazon.com/training/digital/aws-builder-labs/>

- **AWS Immersion Days** – Free, instructor-led virtual events where you follow along with live labs. Check the AWS Events page for schedules.

- **Well-Architected Labs** – Hands-on labs aligned with the AWS Well-Architected Framework pillars.  
  <https://wellarchitectedlabs.com/>

- **Cloud Resume Challenge** – Build a serverless resume website while learning core AWS services.  
  <https://cloudresumechallenge.dev/>

- **Andrew Brown's AWS Project Bootcamp** – A free, project-based bootcamp covering real-world AWS scenarios (available on YouTube and GitHub).

---

> **Tip:** Consider using the **AWS Builder Center Sandbox** for risk-free practice without needing your own AWS account. It is ideal for certification preparation and workshop-based learning. See the [Builder Center Sandbox guide](builder-center-sandbox.md) for details.

---

> **Tip:** The AWS Free Tier is **limited** — it provides 12 months of free usage for select services, plus some always-free offers. Always monitor your billing dashboard to avoid unexpected charges when experimenting beyond Free Tier limits.

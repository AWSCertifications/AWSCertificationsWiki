## What Is AWS Builder Center Sandbox?

The **AWS Builder Center Sandbox** is an official, free, temporary AWS environment that lets you experiment with AWS services without needing your own AWS account. It is part of the [AWS Builder Center](https://aws.amazon.com/builder-center/) and was designed to eliminate the barrier of account setup, billing concerns, and cost anxiety when learning AWS.

Unlike the AWS Free Tier (which runs on your own account), the Builder Center Sandbox provides a pre-configured, isolated environment that is automatically cleaned up after each session. This makes it ideal for workshops, certification practice, and exploring AWS services without any risk of accidental charges.

## Session Details

### Duration

Sessions currently last **up to 8 hours**. This is sufficient for most workshops, certification practice sessions, and hands-on learning activities. If you need more time, you can start a new session (subject to availability and usage limits).

### Weekly Usage Limits

AWS applies reasonable usage limits to ensure fair access for all users. These limits may vary based on demand and workshop availability. Check the [AWS Builder Center](https://aws.amazon.com/builder-center/) for current limits.

### Automatic Cleanup

All resources created during a sandbox session are **automatically destroyed** when the session ends. This means:

- No need to manually clean up resources
- No risk of forgetting to terminate instances
- No lingering costs
- Fresh environment for each session

The Builder Center Sandbox is available through supported workshops on the [AWS Builder Center](https://aws.amazon.com/builder-center/). Workshops cover a wide range of topics including:

- Compute (EC2, Lambda, Auto Scaling)

New workshops are added regularly. Check the [Builder Center](https://aws.amazon.com/builder-center/) for the latest offerings.

## Limitations

- **Session length:** Maximum 8 hours per session
- **Usage limits:** Weekly usage caps apply
- **Workshop-dependent:** Not all services may be available in every workshop
- **No persistence:** All data is lost when the session ends
- **No production use:** Not intended for production workloads or sustained development
- **Limited customization:** Environment is pre-configured for the specific workshop
- **Internet access:** May be restricted within the sandbox environment

## Frequently Asked Questions

**Is the Builder Center Sandbox really free?**
Yes — completely free. No AWS account, credit card, or payment method is required.

**Who can use the Builder Center Sandbox?**
Anyone with access to the [AWS Builder Center](https://aws.amazon.com/builder-center/). There are no eligibility restrictions.

**Do I need an AWS account to use it?**
No. The sandbox provides a temporary AWS account automatically.

**Do I need a Skill Builder subscription?**
No. Access is through the AWS Builder Center, which is separate from Skill Builder.

**How long does a session last?**
Currently up to 8 hours. The exact duration depends on the specific workshop.

**What happens when my session ends?**
All resources are automatically cleaned up and the temporary account is deactivated.

**Can I extend a session?**
You may start a new session if availability allows, subject to usage limits.

**Are there usage limits?**
Yes, AWS applies reasonable weekly limits to ensure fair access.

**Can I use it for exam practice?**
Yes — it is ideal for certification practice, especially for hands-on exam scenarios.

**What services are available?**
Available services depend on the specific workshop. Most core AWS services are supported.

**Can I install custom software?**
Pre-installed tools depend on the workshop. Some workshops include a browser-based terminal with AWS CLI.

**Is my data saved between sessions?**
No. Each session provides a fresh, isolated environment. Export any work before the session ends.

**Can I access the sandbox from anywhere?**
Yes — access is through your browser via the AWS Builder Center.

**How is this different from AWS Free Tier?**
The Free Tier runs on your own account and requires a credit card. The Builder Center Sandbox provides a temporary account with no billing risk.

**How is this different from Skill Builder Labs?**
Skill Builder Labs require a Skill Builder subscription (limited free tier) and are accessed through the Skill Builder platform. The Builder Center Sandbox is accessed through the AWS Builder Center.

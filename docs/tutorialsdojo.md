# TutorialsDojo (Jon Bonso) — Udemy vs TutorialsDojo.com

> Source: [Reddit — u/madrasi2021](https://www.reddit.com/r/AWSCertifications/comments/xyz)

---

## Overview

Jon Bonso's Tutorials Dojo practice exams are widely considered the gold standard for AWS certification prep. However, where you buy them — **Udemy** vs **TutorialsDojo.com** — makes a difference.

---

## Comparison Table

| Feature | Udemy | TutorialsDojo.com |
|---|---|---|
| **Pricing** | Lower in some countries (PPP), lifetime | Direct pricing, 1-year access |
| **Updates** | Slower to receive new questions | Faster, more frequent updates |
| **Review Mode** | Recently added (similar to TD) | Excellent, question-by-question mode |
| **Lifetime Access** | Yes | No (1 year only) |
| **Hands-on Labs** | No | Yes (AWS Sandbox + 100+ guided labs) |
| **Video Course** | No | Yes (exam-specific lectures) |
| **eBooks / Cheat Sheets** | No | Yes |
| **Free AWS Courses** | No | 100+ free courses |

---

## Detailed Breakdown

### Udemy

!!! tip "Pros"
    - **Purchase Price Parity** — Significantly cheaper in countries like India (~$5 USD).
    - **Lifetime Access** — Buy once, access forever, even if content updates later.
    - **Review Mode** — Recently added, parity with the TD website experience.
    - **Free via Libraries** — Some US public libraries offer free Udemy access.

!!! warning "Cons"
    - **Slower Updates** — New questions and fixes take longer to appear.
    - **No Labs / Videos / eBooks** — Only practice exams; supplementary materials are not included.
    - **Lower Author Margin** — Udemy takes a large cut of each sale.

### TutorialsDojo.com

!!! tip "Pros"
    - **Faster Updates** — Questions are refreshed immediately.
    - **Review Mode** — Polished, bite-sized question-by-question learning.
    - **Hands-on Labs** — AWS Sandbox environment + 100+ guided labs.
    - **Video Course** — Full exam-specific lecture series included.
    - **eBooks & Cheat Sheets** — Downloadable study materials.
    - **100+ Free Courses** — Additional free content on the platform.
    - **Better Author Support** — More money goes directly to the creator.

!!! warning "Cons"
    - **1-Year Access Only** — No lifetime purchase; access expires after 12 months.
    - **Higher Upfront Cost** — No PPP pricing; price is the same globally.

---

## Recommendation

- **Buy from Udemy** if you're in a PPP-eligible country, want lifetime access, or can get it free through a library. You'll save money but miss out on labs, videos, and the best review mode.
- **Buy from TutorialsDojo.com** if you want the full experience (labs, videos, cheat sheets), faster updates, and want to support the author directly. The 1-year access window is usually plenty for exam prep.

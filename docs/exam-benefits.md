# AWS Exam Pass Benefit — 50% Off Your Next Exam

Pass **any** AWS certification exam and you earn a **50% discount** on any other AWS certification exam. This is a recurring benefit tied to each active certification you hold.

!!! tip "Last Updated"
    This page was last updated on **23 June 2026**.

## How It Works

- Pass an AWS exam → get one **50% off** voucher
- One voucher **per exam passed**
- Valid for **any** AWS exam (Practitioner, Associate, Professional, Specialty)
- Voucher expires **3 years** from issue (as long as the certification remains active)

## FAQ

| Question | Answer |
|----------|--------|
| Does this apply to any AWS exam? | Yes |
| Do I get this if I fail? | No |
| Where do I find the voucher? | [aws.training](https://aws.training) → Certification → Sign In → Expand **"BENEFITS"** → Certification benefits |
| When do vouchers expire? | **3 years** — active as long as the certification is valid |
| Does a BETA exam count? | Yes |
| Can I use a Practitioner voucher for a Professional/Specialty exam? | Yes — valid for any AWS exam |
| Can I stack it with another discount? | No — only one voucher per exam booking |
| Can I sell or transfer my benefit? | **No.** Benefits are non-transferable. Violation may invalidate your exam results |
| Used the voucher and failed — can I reuse? | No — single use only |
| Cancelled the exam — can I reuse? | Yes, if the benefit hasn't expired |
| My certification expired — can I still use the voucher? | No |
| What if I pass multiple exams? | You get one benefit per exam passed. You cannot stack two benefits for 100% off, but having multiple means you have a backup voucher available |
| Where can I find other discounts? | See [Vouchers & Discounts](vouchers-discounts.md) for other discount options |

!!! warning
    - One voucher per exam booking — cannot combine with other promotions
    - Benefits are non-transferable — selling or transferring may invalidate your exam results
    - Voucher expires if the underlying certification expires

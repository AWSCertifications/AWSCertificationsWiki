# AWS Microcredentials

AWS provides an **expanding collection** of official Skill Builder Microcredentials covering multiple AWS domains. These are short, focused credentials designed to validate practical, hands-on skills through [AWS Skill Builder](https://skillbuilder.aws/).

AWS regularly adds new microcredentials to address evolving cloud skill areas.

!!! warning "Not full certifications"
    Microcredentials are **not replacements** for AWS Certifications. They are narrower in scope and focus on specific hands-on tasks rather than broad architectural knowledge.

## What They Cover

AWS Microcredentials span a wide and growing range of domains, including:

- **Cloud Foundations** — Foundational cloud concepts and AWS services
- **Architecture** — Designing and building AWS architectures
- **Security** — Security best practices and AWS security services
- **Networking** — Networking fundamentals and AWS networking services
- **Machine Learning** — ML/AI concepts and applying AWS AI services
- **Generative AI** — Building and deploying generative AI applications
- **Containers** — Container orchestration and management on AWS
- **Serverless** — Serverless application development and deployment
- **Data & Analytics** — Data processing, analysis, and visualization
- **Storage** — AWS storage services and data lifecycle management

!!! tip "Hands-on validation"
    Unlike some multiple-choice exams, microcredentials require you to complete **practical tasks** in the AWS console, proving you can apply what you've learned.

## How to Earn One

1. Go to [skillbuilder.aws](https://skillbuilder.aws/)
2. Sign in with your Amazon account (free)
3. Search for **"microcredentials"** in the catalog
4. Enroll and complete the learning plan
5. Pass the hands-on assessment

!!! info "Prerequisites"
    Some microcredentials benefit from prior AWS experience. Review the learning plan before enrolling to ensure you meet any recommended prerequisites.

## Why Earn Microcredentials?

- **Free** — no exam fees
- **Practical** — validate hands-on skills, not just theory
- **Fast** — typically take a few hours to complete
- **Complementary** — pair them with certification prep to reinforce real-world skills

These programs complement AWS Certifications and Microcredentials but are **not** certifications themselves.

## FAQ

### What are AWS Microcredentials?

AWS Microcredentials are short, focused, hands-on credentials available through AWS Skill Builder. Each microcredential includes a learning plan and a practical assessment that validates your ability to work with specific AWS services and concepts.

### Are Microcredentials AWS Certifications?

No. Microcredentials are not AWS Certifications. They are narrower in scope, focused on validating practical hands-on skills rather than broad architectural or technical knowledge. They complement certifications but do not replace them.

### Should I complete Microcredentials before taking an AWS Certification?

Microcredentials can be a useful stepping stone, especially if you are new to AWS. They provide hands-on practice that reinforces concepts covered in certification exams. However, they are not a prerequisite — you can pursue certifications directly.

### How many AWS Customer Learning Program badges are available?

AWS currently offers **30+ official Customer Learning Program badges** and continues expanding the collection. New programs are added regularly as AWS releases new services and solution areas.

### Are these badges free?

Many AWS Customer Learning Program badges are available through AWS Skill Builder or official AWS learning programs, although availability depends on the specific program. Some may require a Skill Builder subscription or participation in specific AWS programs.

### Do employers value them?

These badges demonstrate continuous learning and practical engagement with AWS services. They complement AWS Certifications by showing hands-on familiarity with specific services and solution areas, but they do not replace an AWS Certification for roles that require broad architectural or technical expertise.

# AWS Learning Programs & Badges

AWS offers multiple official learning programs beyond certifications. Each program serves a different purpose and awards its own credential. This page explains the differences.

## AWS Skill Builder Microcredentials

AWS Skill Builder Microcredentials are official [AWS Skill Builder](https://skillbuilder.aws/) learning paths that validate topic-focused cloud skills. Each microcredential includes structured training and a final hands-on assessment that requires completing practical tasks in the AWS console.

Key characteristics:

- **Topic-focused** — Each microcredential covers a specific AWS domain (security, networking, serverless, etc.)
- **Hands-on assessment** — Pass a practical task in the AWS console, not a multiple-choice exam
- **Digital badge** — Earn an AWS digital badge upon completion
- **Complement certifications** — Microcredentials are not replacements for AWS Certifications. They are narrower in scope and focus on specific hands-on tasks rather than broad architectural knowledge.

[Learn more](https://skillbuilder.aws/certification/microcredentials)

## AWS Customer Learning Programs

AWS Customer Learning Programs are official AWS learning programs focused on individual AWS services and solution areas. Each program awards a Credly digital badge and helps learners deepen practical knowledge of specific AWS services beyond the certification blueprint.

Key characteristics:

- **Service-specific** — Focus on individual AWS services, products, and solution areas
- **Credly badge** — Earn a Credly digital badge that can be shared on LinkedIn and social platforms
- **Practical knowledge** — Designed to build hands-on familiarity with specific AWS services
- **Not certifications** — These programs complement AWS Certifications but are not certifications themselves

AWS currently offers 30+ Customer Learning Program badges and continues expanding the collection.

[Explore programs](https://www.credly.com/organizations/amazon-web-services/collections/customer-learning-programs/badge_templates)

## AWS SimuLearn Learning Plan Badges

AWS SimuLearn Learning Plan Badges are official AWS credentials that validate practical cloud skills through interactive simulations. Unlike multiple-choice exams, SimuLearn immerses you in realistic scenarios — including AI-powered customer conversations, architecture design, hands-on building in live AWS environments, and automated solution validation.

SimuLearn Learning Plan Badges are **separate** from AWS Certifications, AWS Skill Builder Microcredentials, and AWS Customer Learning Programs. They represent a different approach to skill validation.

Key characteristics:

- **Interactive simulations** — Engage with virtual customers, design architectures, and build solutions in real AWS environments
- **End-to-end validation** — Complete all assignments in a learning plan to earn the badge
- **AWS digital badge** — Verifiable credential issued through Credly
- **Certification preparation** — Excellent hands-on practice before taking AWS Certification exams

### Availability

AWS currently offers **12 SimuLearn Learning Plan Badges**. Cloud Practitioner and AI Practitioner learning plans are currently free. The remaining learning plans require an AWS Skill Builder Individual or Team subscription.

!!! info "Availability may change"
    Availability, pricing, and the number of learning plans may change over time.

[Learn more](https://aws.amazon.com/blogs/training-and-certification/introducing-aws-simulearn-learning-plan-badges-verifiable-proof-of-hands-on-cloud-skills/)

## Frequently Asked Questions

### AWS Skill Builder Microcredentials

**What are AWS Skill Builder Microcredentials?**

AWS Skill Builder Microcredentials are short, focused, hands-on credentials available through AWS Skill Builder. Each microcredential includes a learning plan and a practical assessment that validates your ability to work with specific AWS services and concepts.

**Are Microcredentials AWS Certifications?**

No. Microcredentials are not AWS Certifications. They are narrower in scope, focused on validating practical hands-on skills rather than broad architectural or technical knowledge. They complement certifications but do not replace them.

**Should I complete Microcredentials before taking an AWS Certification?**

Microcredentials can be a useful stepping stone, especially if you are new to AWS. They provide hands-on practice that reinforces concepts covered in certification exams. However, they are not a prerequisite — you can pursue certifications directly.

**Do employers value Microcredentials?**

These badges demonstrate continuous learning and practical engagement with AWS services. They complement AWS Certifications by showing hands-on familiarity with specific services and solution areas, but they do not replace an AWS Certification for roles that require broad architectural or technical expertise.

### AWS Customer Learning Programs

**How many AWS Customer Learning Program badges are available?**

AWS currently offers **30+ official Customer Learning Program badges** and continues expanding the collection. New programs are added regularly as AWS releases new services and solution areas.

**Are Customer Learning Program badges free?**

Many AWS Customer Learning Program badges are available through AWS Skill Builder or official AWS learning programs, although availability depends on the specific program. Some may require a Skill Builder subscription or participation in specific AWS programs.

**Do employers value Customer Learning Program badges?**

These badges demonstrate continuous learning and practical engagement with AWS services. They complement AWS Certifications by showing hands-on familiarity with specific services and solution areas, but they do not replace an AWS Certification for roles that require broad architectural or technical expertise.

### AWS SimuLearn Learning Plan Badges

**What are AWS SimuLearn Learning Plan Badges?**

AWS SimuLearn Learning Plan Badges are verifiable AWS digital badges awarded for completing hands-on SimuLearn learning plans. They validate practical cloud skills through interactive simulations, customer conversations, and live AWS environment exercises.

**Are SimuLearn badges certifications?**

No. SimuLearn badges are not AWS Certifications. They are credentials that validate completion of hands-on learning plans. They complement certifications but do not replace them.

**Are SimuLearn badges free?**

Cloud Practitioner and AI Practitioner SimuLearn learning plans are currently free. The remaining 10 learning plans require an AWS Skill Builder Individual or Team subscription.

**Should I complete SimuLearn before taking an AWS Certification?**

SimuLearn learning plans are optional but provide excellent hands-on preparation. They are particularly useful for reinforcing practical skills before attempting an AWS Certification exam, especially at the Foundational and Associate levels.

**Do SimuLearn badges replace AWS Certifications?**

No. SimuLearn badges complement AWS Certifications by validating practical cloud skills. AWS Certifications remain the primary credentials for demonstrating broad AWS knowledge and expertise.

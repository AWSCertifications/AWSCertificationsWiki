# AWS Microcredentials

AWS offers **5 official microcredentials** — short, focused credentials designed to validate practical, hands-on skills. These are available for **free** through [AWS Skill Builder](https://skillbuilder.aws/).

!!! warning "Not full certifications"
    Microcredentials are **not replacements** for AWS Certifications. They are narrower in scope and focus on specific hands-on tasks rather than broad architectural knowledge.

## What They Cover

Each microcredential targets a practical skill area:

- **AWS Cloud Essentials** – Foundational cloud concepts and AWS services
- **AWS Architecting** – Designing and building basic AWS architectures
- **AWS Machine Learning** – ML/AI concepts and applying AWS AI services
- **AWS Security** – Security best practices and AWS security services
- **AWS Networking** – Networking fundamentals and AWS networking services

!!! tip "Hands-on validation"
    Unlike some multiple-choice exams, microcredentials require you to complete **practical tasks** in the AWS console, proving you can apply what you've learned.

## How to Earn One

1. Go to [skillbuilder.aws](https://skillbuilder.aws/)
2. Sign in with your Amazon account (free)
3. Search for **"microcredentials"** in the catalog
4. Enroll and complete the learning plan
5. Pass the hands-on assessment

!!! info "Prerequisites"
    Some microcredentials benefit from prior AWS experience. Review the learning plan before enrolling to ensure you meet any recommended prerequisites.

## Why Earn Microcredentials?

- **Free** — no exam fees
- **Practical** — validate hands-on skills, not just theory
- **Fast** — typically take a few hours to complete
- **Complementary** — pair them with certification prep to reinforce real-world skills
